- `index.html` – Startdatei für GitHub Pages
- `GenussWerk_App_2026_V146_Dienstplan_Personal_Sync.html` – Backup der originalen Versionsdatei

## Wichtig

Diese App ist aktuell eine reine Browser-/HTML-App. Alles, was in der Datei steht, kann bei einem öffentlichen Repository grundsätzlich eingesehen werden.

Für echte Personaldaten, Passwörter oder produktive Nutzung sollte das Repository privat bleiben oder später eine echte Benutzerverwaltung mit Server/Backend eingesetzt werden.
